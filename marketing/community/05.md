# 에브리타임

제목: 무료 키워드 분석기 만들었어요 (대학생 블로거분들께 도움 될 것 같아서)

---

안녕하세요! 저도 학교 다니면서 블로그로 용돈 벌려고 공부하다가 만든 도구예요.

wealthpipe.net 인데요, 네이버 블로그 키워드 분석을 무료로 할 수 있어요. 비싼 키워드 도구 구독 안 해도 되고 로그인도 없어요.

기능은 이런 것들이에요.

- 키워드 검색량이랑 경쟁도 한눈에 확인
- 기회점수로 공략하기 쉬운 키워드 추천
- AI가 글 구조 잡아줌 (주제 잡기 어려울 때 유용해요)
- 12개월 트렌드로 계절 키워드 파악

학교 다니면서 블로그 운영하시는 분들, 수익화 시작하려는 분들한테 도움이 될 것 같아요.

써보시고 개선 의견 있으시면 알려주세요. 같이 발전시켜 나가고 싶어요!
