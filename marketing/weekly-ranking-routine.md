## 3. 매주 월요일 루틴 체크리스트

### 준비 단계 (오전 9시 이전)

- [ ] WealthPipe 접속 → 주간 키워드 랭킹 TOP 20 확인
- [ ] 전주 랭킹과 비교해 순위 변동 체크 (▲▼ NEW 표시)
- [ ] 이번 주 주목 키워드 1개 선정 (신규진입 or 급상승 우선)
- [ ] 선정 키워드 관련 최근 이슈 간단히 검색 (1~2분)

### 블로그 발행 (오전 9~10시)

- [ ] 제목 작성: "이번 주 인기 키워드 TOP 10 (N월 N주차)"
- [ ] 본문 템플릿에 이번 주 데이터 채워 넣기
- [ ] 주목 키워드 분석 2~3문장 작성
- [ ] CTA 링크(wealthpipe.net) 확인
- [ ] 해시태그 추가 후 발행

### 카카오톡 공유 (블로그 발행 직후)

- [ ] 카카오톡 메시지 템플릿에 이번 주 데이터 입력
- [ ] 블로그 포스트 URL 복사
- [ ] 공유할 채널/방에 메시지 + 블로그 URL 함께 전송

### 마무리

- [ ] 블로그 발행 확인 (모바일로 한 번 확인)
- [ ] 카카오톡 메시지 발송 확인
- [ ] 다음 주를 위해 이번 주 TOP 5 메모해두기 (변동 추적용)

---

## 작업 시간 기준

전체 소요 시간: 약 20~30분

- 데이터 확인 및 키워드 선정: 5분
- 블로그 포스트 작성 및 발행: 15~20분
- 카카오톡 공유: 2~3분

---

## 주차 계산 기준

N월 N주차 = 해당 월의 몇 번째 월요일인지 기준

예시:
- 4월 첫 번째 월요일 → 4월 1주차
- 4월 두 번째 월요일 → 4월 2주차
